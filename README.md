
# DeFinE

Decentralized Finance Enhanced


## For Local Development

 Clone the Repo and Install all the dependencies


```bash
    cd client 
    yarn 
```

```bash
    cd backend 
    yarn 
```

```bash
    cd ekycbackend 
    yarn 
```

Run all the backend services and Start the Frontend server


## Environmental Variables

- Moralis API Key 
- MongoDB Atlas URI
